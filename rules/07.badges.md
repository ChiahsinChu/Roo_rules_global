# Badges

Before generating badges for [tests](#tests) and [coverage](#test-coverages), run test first:

```bash
mkdir -p docs/reports
mkdir -p docs/badges
pytest  --cov=<pkgs> tests \
--junitxml=junit.xml --html=docs/reports/test.html \
--cov-report xml \
--cov-report html:docs/reports/coverage
```

## tests

```bash
genbadge tests -i junit.xml -o ./docs/badges/tests.svg
```

Usage example for markdown:

```markdown
[![Tests Status](./docs/badges/tests.svg?dummy=8484744)](./docs/reports/test.html)
```

## test coverages

```bash
genbadge coverage -i coverage.xml -o ./docs/badges/coverage.svg
```

Usage example for markdown:

```markdown
[![Coverage Status](./docs/badges/coverage.svg?dummy=8484744)](./docs/reports/coverage/class_index.html)
```

## docstrings coverages

```bash
docstr-coverage src/<pkgs>/ \
    --skip-private \
    --skip-property \
    --accept-empty \
    --exclude=".*/_version.py" \
    --badge=./docs/badges/docstring-coverage.svg
```

Usage example for markdown:

```markdown
[![Docstring Coverage](./docs/badges/docstring-coverage.svg?dummy=8484744)](https://github.com/HunterMcGushion/docstr_coverage)
```
